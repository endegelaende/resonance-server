# Resonance Web UI

Modern web interface for the Resonance music server, built with Svelte 5, SvelteKit, and Tailwind CSS v4.

For full documentation, installation instructions, and feature overview see the
[main README](../README.md#web-ui).

## Quick Start

Make sure the Resonance Python backend is running first (see [Starting the Server](../README.md#starting-the-server)).

```bash
npm install       # one-time
npm run dev       # http://localhost:5173
```

## Commands

```bash
npm run dev       # development server with hot reload
npm run build     # production build (output in build/)
npm run preview   # preview production build
npm run check     # type checking
```
